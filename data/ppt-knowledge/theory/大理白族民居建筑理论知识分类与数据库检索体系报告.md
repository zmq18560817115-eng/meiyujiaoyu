# 大理白族民居建筑理论知识分类与数据库检索体系报告

## 1. 引言：白族建筑空间的战略地位与知识建构意义

大理白族民居作为横断山脉地理适应性与中原儒家伦理深度融合的产物，具有极高的文化战略意义。其建筑逻辑不仅在物理层面解决了滇西高海拔峡谷地带的防风、采光与立体排水需求，更在空间层面通过严谨的形制具象化了宗族社会的礼制秩序。建立该建筑理论的知识分类体系，旨在为数字化时代的遗产保护提供标准化的逻辑底层，确保建筑数据库在数字化检索、三维建模及当代设计转译过程中具备严密的属性关联与元数据支撑。

## 2. 空间布局体系：从功能单元到社会经济分层的分类检索

白族民居的形态结构（Morphological Structure）是地理约束与社会逻辑的复合体。其布局形式直接反映了家族的社会经济分层（Socio-economic Stratification），通过正房体量与石质基座的高度差异，明晰了儒家「长幼有序」的等级秩序。

| 布局名称 | 形态结构与组合方式 | 核心功能机制 | 社会经济分层 |
| ------ | ------ | ------ | ------ |
| 一正两耳 | 一幢两层正房带两间低矮耳房 | 基础居住与储藏，结构紧凑 | 经济一般、人口较少的普通农户 |
| 三坊一照壁 | 三坊楼房呈「∏」形，正房对设独立照壁 | 照壁挡西风并利用白色墙面反射阳光入堂屋 | 典型中产阶级及以上，分布最广的经典合院 |
| 四合五天井 | 四坊楼房围合，中心大天井配四角「漏角天井」 | 漏角天井兼顾辅助采光、垂直排水及局部功能分隔 | 实力雄厚的商贾或士绅，寓意「四方聚财」 |
| 六合同春 | 一进数院的复合型群落，多院落纵横贯通 | 削弱大体量建筑压迫感，通过回廊实现全域通达 | 地方巨商或显贵，如喜洲严家、杨家等大院 |
| 走马转角楼 | 四坊楼房在二层通过通长外廊相互连通 | 便于雨季楼层穿行，通过物理连通增强高层安全性 | 多见于降水充沛、资金充裕的大家族 |

**漏角天井的空间转换价值：** 在大型合院中，漏角天井是实现空间属性定义的关键节点。它位于两坊建筑垂直相交处，作为垂直排水系统的枢纽，同时为深宅内部提供了必要的呼吸孔隙。

## 3. 营造工艺与材料系统：物质文化资产的特征提取

「石」与「木」在大理建筑中并非单纯的建筑材料，而是对抗自然灾害的战略工具。「大理有三宝，石头砌墙不会倒」不仅是谚语，更是其石构抗震技术的真实写照。

### 物质构造的属性定义

- **石材分工**：海东青山石作为核心材料，在墙基、门头、横梁中承担着主要的刚性支撑。
- **木构力学**：斗拱出挑实现了屋檐力学的平衡与美学的外延。「六合门」系统通过榫卯结构实现可拆卸、可完全开启的物理状态，是堂屋与庭院虚实转换的核心开关。

### 关键材料库特征词

- **海东青山石**：高耐候性，主要用于精密凿刻的门头基座。
- **粘土青瓦**：坡度精准，配合天井形成科学的径流控制。
- **熟桐油与猪血石灰浆**：用于木构件打底（油灰），形成抗腐蚀保护膜。
- **纸筋灰**（熟石膏与白棉纸拌合）：墙面彩绘底料。核心工艺约束：若当天混合的底料未用完，必须在收工前彻底刮除并在次日重新制作，这是确保彩绘永久附着的技术红线。

## 4. 符号学与伦理体系：照壁家风与山花纹样的关键词分类

白族民居的墙面是承载家族信息的「媒介资产」。照壁题字与山花纹样通过特定的语义关联，实现了建筑空间的身份验证与教化功能。

### 核心关键词索引（姓氏-题字-典故映射）

- **杨姓**：清白传家 / 清白世家（典出杨震「暮夜却金」）
- **张姓**：百忍家风 / 百忍家声（典出张公艺「九代同堂」）
- **李姓**：青莲遗风 / 青莲世第（典出李白，劝导学优入仕）
- **赵姓**：琴鹤家声 / 琴鹤遗风（典出赵抃「一琴一鹤」，寓意官清似水）
- **王姓**：三槐流芳 / 瑞雪三槐（典出王祜「手植三槐」，寓意德泽后世）
- **刘姓**：禄阁传芳 / 汉室遗风（典出刘向在天禄阁校书）
- **董姓**：南诏宰辅（典出家族在南诏政权的显赫功绩）

### 山花纹样与语义分类

- **生物类**：龙凤（龙凤呈祥）、蝙蝠（五福临门）、鹤（长寿）
- **植物类**：莲花与三节藕（连续升华 / 三级跳升）、牡丹（富贵）
- **人文类**：暗八仙（宗教祈愿）、琴棋书画（文人情操）

**科学贡献：湿壁画法（Wet Fresco）** 白族彩绘采用「半干绘制工艺」。矿物颜料在纸筋灰底料半干时与其发生化学结合，确保建筑表面色彩历经百年风蚀仍保持极高色牢度。

## 5. 现代演进分析：从传统原型到「向内微合院」的转换逻辑

当代设计（如边缘计划建筑工作室的「向内的微合院」）在现代语境下对传统原型进行了深刻重构。

- **场地约束**：通过「向内看」的内向型策略消解周边混杂商业环境。
- **空间消解**：退让建筑界面、增加虚空界面（露角天井），消解两层现代楼房对庭院的压迫感。
- **克制开窗与取景**：外立面克制开窗，捕捉崇圣寺千寻塔、苍山山脊与邻里石墙。
- **漏角天井的当代转译**：转译为现代客房私密内庭，保留通气排水功能，实现混凝土、金属、玻璃与传统水墨审美共生。
