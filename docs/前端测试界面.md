## 一、打开测试界面

在项目根目录执行：

```bash
cd "/Users/zhangmingqi/Desktop/人机交互设计/untitled"
npm install
npm run dev
```

浏览器访问（前后端同一端口）：

| 用途 | 地址 |
|------|------|
| **主测试入口** | **http://localhost:3000** |
| 教师名册 API | http://localhost:3000/api/auth/roster |
| 健康检查 | http://localhost:3000/api/health（若已实现） |

生产构建预览：

```bash
npm run build
npm start
```

同样访问 **http://localhost:3000**（静态资源由 Express 托管 `dist/`）。

## 六、常见问题

| 现象 | 处理 |
|------|------|
| 无法连接 localhost:3000 | 终端执行 `npm run dev`，确认无端口占用 |
| 页面是旧版（无小茶立绘） | `Cmd + Shift + R` 强制刷新 |
| 小茶仍是方框底图 | 确认存在 `src/assets/xiaocha-character-cutout.png` |
| AI 无真实回复 | 未配置 `GEMINI_API_KEY` 时使用离线兜底文案，属正常 |
| 端口被占用 | `lsof -ti:3000 \| xargs kill -9` 后重新 `npm run dev` |
